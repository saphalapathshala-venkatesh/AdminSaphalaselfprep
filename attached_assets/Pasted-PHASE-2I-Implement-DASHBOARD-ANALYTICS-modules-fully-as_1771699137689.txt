2) GET /api/analytics/report

3) GET /api/analytics/export
Query same as report
Return CSV download (set correct headers)

IMPLEMENTATION NOTES:
- Use start/end inclusive range; treat end as end-of-day (23:59:59)
- LearnerFilter:
   paid = users with any ACTIVE entitlement
   free = users with zero ACTIVE entitlements
- Stream filter affects Purchase.stream for revenue charts and KPIs.
- Revenue net should use Purchase.netPaise already stored.
- Active users/day:
   count distinct userId from Attempt.startedAt within day
   plus optionally from Session.createdAt within day (ADMIN sessions should not count; only STUDENT sessions). If you track Session.type, count only STUDENT.
- Top XP earners:
   sum XpEvent.points by user in range, order desc
- Recently published content:
   union of ContentPage and PdfAsset where isPublished=true, order by publishedAt desc, take 10
