B) Learners API: /api/learners
1) GET /api/learners
   - Query: page,pageSize,search (name/email/mobile), filter paid/free/all
   - Paid/free:
     paid = users who have any ACTIVE entitlement (UserEntitlement.status=ACTIVE)
     free = no active entitlements
   - Return list with:
     user id, name, email, mobile, role, isActive
     totalXp (computed from XpLedgerEntry sum OR cached if available)
     lastActiveAt (use latest Attempt.startedAt OR Session.createdAt as fallback)
     activeEntitlements count
2) GET /api/learners/[id]
   - Return profile:
     user basics
     entitlements list (productCode, status, validUntil)
     last 20 attempts (testId, scorePct, startedAt)
     XP history:
       last 50 XpLedgerEntry rows (delta, reason, createdAt)
     purchases summary (last 10 purchases)
3) PUT /api/learners/[id]/entitlements
   - SUPER_ADMIN only

     upsert entitlement for (userId, productCode, tenantId="default"):
       status ACTIVE, validUntil, purchaseId null
   - REVOKE:
     find entitlement and set status REVOKED (do not delete)
   - Audit: LEARNER_ENTITLEMENT_GRANT / LEARNER_ENTITLEMENT_REVOKE with before/after

4) PUT /api/learners/[id]/status
