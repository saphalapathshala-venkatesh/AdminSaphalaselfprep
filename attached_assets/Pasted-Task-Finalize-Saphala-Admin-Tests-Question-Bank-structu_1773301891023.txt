PART 5 — DELETION GOVERNANCE RULES
==================================================

Implement these deletion rules safely and clearly.

A. Test Series deletion
- A test series can be deleted ONLY if it has no published tests
- If published tests exist, block deletion
- Show a clear message explaining why deletion is blocked

B. Test deletion
- Tests can be deleted
- Deleting a test must NOT delete underlying questions from Question Bank
- It should only remove the test structure / associations safely

C. Section deletion
- Sections can be deleted from a test
- Deleting a section must NOT delete underlying bank questions
- It should only remove section/test structure or section-question associations as appropriate

D. Subsection deletion
- Subsections can be deleted from a test
- Deleting a subsection must NOT delete underlying bank questions
- It should only remove subsection/test structure or subsection-question associations as appropriate

E. Question deletion rule
- Questions must be deleted ONLY through the Question Bank
- Do NOT allow destructive question deletion from tests / sections / subsections / test builder
- In test builder, use wording like:
  - Remove from Test
  - Remove from Section
  - Remove from Subsection
  not “Delete Question”

This distinction is very important.
