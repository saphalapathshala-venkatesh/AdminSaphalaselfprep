PART 4 — DOWNLOADABLE CONTROL FOR PDF COURSES AND INDIVIDUAL PDFS
==================================================

Implement download control at two levels:

A. Course-level PDF download control
- When creating/configuring a PDF course, admin must be able to choose:
  - Downloadable = Yes
  - Downloadable = No
- This should control whether PDFs in that course are intended to be downloadable by learners

B. Individual PDF-level download control
- Each individual PDF should also have its own Downloadable Yes/No control
- This allows finer control per document
- Example:
  - Course may be protected overall
  - But one specific PDF may still be allowed or blocked
- Or vice versa if your current architecture resolves effective access this way

Important:
- The system should clearly allow/prevent download based on admin settings
- Implement admin-side data/control now; student-side enforcement should remain compatible with this rule
- Avoid ambiguous mixed behavior
