1) Cloze / Blank Maker Tool

Add an editor tool that lets the admin quickly convert selected text into a blank / cloze placeholder.

Purpose

Useful for:

fill-in-the-blank learning content

revision notes

flashcard practice

exam-style reinforcement

Expected behavior

Admin selects a word or phrase in the editor

Clicks Blank / Cloze tool

The selected text is converted into a blank placeholder visually in the editor

Original answer data should still be retained in structured form where needed for future use

If implemented first only as a content-authoring helper, ensure rendering is stable and visually clear

UI expectation

Toolbar button label can be something like:

Blank

Cloze

Fill Blank

Example

Input:
Fundamental Rights are enforceable in courts.

After selecting enforceable and clicking the tool:
Fundamental Rights are ______ in courts.

Important

Do not make this fragile or purely visual if the editor can store a structured representation safely

If structured representation is too risky right now, implement a safe first version that behaves predictably in stored content
