CRITICAL PRODUCT RULES

1. Questions will NOT contain Section or Subsection columns in bulk templates.
   Reason: Admin selects section/subsection before uploading questions.

2. Upload workflow must work like this:
   Admin selects a Section
   → If subsection exists, selects Subsection
   → Clicks "Add Questions"
   → Options appear:
      - Add Single Question
      - Bulk Upload Questions

3. Bulk upload templates must therefore NOT contain fields:
   - Section
   - Subsection

4. Marks must remain configurable per question.
   Different questions may carry different marks.

5. Negative marks remain optional per question.

6. Timer configuration in tests should be entered in MINUTES (not seconds).

7. During the exam runtime, the timer display must be formatted as:

   HH:MM:SS

   Example:
   90 minutes input → displayed as 01:30:00
